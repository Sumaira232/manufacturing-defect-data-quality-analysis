# 📌 Project Overview

This project evaluates the quality of a manufacturing defect dataset by identifying data inconsistencies, missing values, duplicates, formatting issues, and validation errors. The findings are documented through a comprehensive report, executive summary, and walkthrough video.

---

# 🎯 Objectives

- Assess overall data quality
- Detect missing values
- Identify duplicate records
- Validate data consistency
- Improve data reliability
- Prepare professional documentation
